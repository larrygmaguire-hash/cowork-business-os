# Platform Specifications

Character limits, format conventions, and best-practice lengths for common content platforms. Verify current limits against the platform's docs -- these change.

## Social

| Platform | Post Limit | Recommended Length |
|----------|-----------|-------------------|
| LinkedIn | 3,000 chars | 1,200-1,600 chars for engagement |
| X / Twitter | 280 chars (free), 25,000 (premium) | 100-200 chars for replies, up to limit for threads |
| Bluesky | 300 chars | Full length usable |
| Threads | 500 chars | 250-400 chars |
| Instagram caption | 2,200 chars | First 125 chars visible before "more" -- front-load |
| Facebook post | 63,206 chars | Under 80 chars for highest engagement |
| TikTok caption | 2,200 chars | 100-300 chars |
| YouTube description | 5,000 chars | First 150 chars show in search |
| YouTube Shorts | 100 chars title, 5,000 desc | |

## Email

| Component | Target |
|-----------|--------|
| Subject line | Under 50 chars (mobile cutoff around 30-40) |
| Preview text | 85-100 chars |
| Body (marketing) | 50-125 words for single-CTA emails |
| Body (newsletter) | 500-1,000 words typical |

## Web

| Component | Target |
|-----------|--------|
| Page title (SEO) | 50-60 chars |
| Meta description | 150-160 chars |
| H1 | One per page, 20-70 chars |
| Hero headline | 6-12 words |
| CTA button | 2-5 words, action verb first |
| Blog post | 1,200-2,500 words for depth; 800-1,200 for commentary |
| Landing page | As long as needed -- skim-readable, scan-first design |

## Ads

| Platform | Headline | Description |
|----------|----------|-------------|
| Google Search | 30 chars × 15 | 90 chars × 4 |
| Google Display | 30 chars | 90 chars |
| LinkedIn single-image | 150 chars (intro) | 70 chars (headline) |
| Meta (Facebook/Instagram) | 40 chars (headline) | 125 chars (primary text) |
| TikTok | 100 chars | 100 chars |
