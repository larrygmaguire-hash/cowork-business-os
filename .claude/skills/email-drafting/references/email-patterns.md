# Email Patterns

Common email formats with structural templates. Adapt tone to match `.claude/company/voice.md`.

## Cold Outreach

```
Subject: [Specific, relevant hook -- not generic]

Hi [Name],

[One line showing you know who they are -- a recent post, role, company move]

[One line on why you are writing -- the specific problem or opportunity]

[One line on the proposed next step -- low-friction ask]

[Your name]
```

Keep under 80 words. No attachments. No "I hope this finds you well."

## Follow-Up (after meeting)

```
Subject: Following up -- [topic from meeting]

Hi [Name],

Thanks for the time today. Summarising what we agreed:

- [Decision/action 1] -- [owner] by [date]
- [Decision/action 2] -- [owner] by [date]
- [Decision/action 3] -- [owner] by [date]

[Any open question]

[Your name]
```

## Proposal Cover Email

```
Subject: Proposal -- [Project name]

Hi [Name],

Attached is the proposal for [project]. Key points:

- [Scope in one line]
- [Timeline in one line]
- [Investment in one line]

Happy to walk through it on a call if helpful. Otherwise, [next step and deadline].

[Your name]
```

## Handling an Objection

```
Subject: Re: [their subject]

Hi [Name],

[Acknowledge the objection directly -- no dismissal, no hedging]

[Address it with specifics -- a fact, an example, a clarification]

[Reframe or offer a path forward]

Happy to discuss if this raises more questions.

[Your name]
```

## Decline

```
Subject: Re: [their subject]

Hi [Name],

Thanks for [the opportunity / the enquiry / considering us].

[Direct, honest reason -- capacity, fit, timing]

[Optional: referral to someone who might suit / offer to reconsider if circumstances change]

[Your name]
```

## Update to Stakeholders

```
Subject: [Project] update -- week of [date]

Hi all,

**Progress this week:**
- [Item 1]
- [Item 2]

**Blockers:**
- [Item or "none"]

**Next week:**
- [Item 1]
- [Item 2]

Let me know if any questions.

[Your name]
```

## Apology

- Acknowledge what happened -- do not minimise
- State what went wrong without excuses
- State what you are doing to resolve it
- State what you are doing to prevent a repeat
- No "I'm sorry you felt that way" constructions. Apologise for the action, not their reaction.

## Subject Line Rules

- Specific beats clever. "Q3 invoice -- due 15 Oct" beats "Important update."
- Front-load the key information. Mobile clients truncate around 30-40 chars.
- Skip "Urgent" unless it genuinely is. Readers tune it out.
- Match the body. A subject that over-promises gets filed as spam next time.
