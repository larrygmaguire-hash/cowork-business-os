# Docs

Reference documentation for users of this workspace. Read these on demand.

## Files

| File | Topic |
|------|-------|
| `getting-started.md` | First-time setup walkthrough |
| `folder-structure.md` | What each folder is for |
| `available-automations.md` | Full skills catalogue with trigger phrases |
| `capabilities-reference.md` | What Cowork can and cannot do in this template |

## How to use

These files are reference material, not auto-loaded. Read them when you need them. The `/setup` skill points users here for deeper understanding.

## When to update

Update these when:
- The template gets new skills (update `available-automations.md`)
- The folder structure changes (update `folder-structure.md`)
- Cowork capabilities change (update `capabilities-reference.md`)
- New terminology is introduced (update `glossary.md`)

If you fork or customise the template substantially, update these to reflect your changes.
