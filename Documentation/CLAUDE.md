# Documentation

Shared reference materials, templates, and reports that span multiple projects or apply across the workspace.

## Subfolders

- **Templates/** -- reusable document templates (proposal templates, report templates, brief templates)
- **Reports/** -- generated reports and analyses (status reports, monthly summaries, ad hoc analyses)

## What goes here

- Reusable templates that any project can pull from
- Cross-project reports and summaries
- Reference documents that apply workspace-wide
- Generated outputs that do not belong to a single project

## What does NOT go here

- Project-specific work (use `Projects/P###/` instead)
- Client-specific documents (use `Clients/[Client]/` instead)
- Department-specific procedures (use the relevant `## Department/` folder)
- Archived completed projects (use `Archive/` instead)
