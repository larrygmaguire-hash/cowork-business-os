## How to use this CLAUDE.md when onboarding a real client

When creating a new client (via `/creating-clients`), the skill generates this CLAUDE.md and asks you for:

- Client name and industry
- Primary contacts (name, role, email)
- Communication preferences (channel, tone, response time)
- Engagement type and rough scope
- Any specific conventions for this client

Update the file as the engagement develops -- new contacts, new projects, key decisions, anything Cowork should know when working on this client.
