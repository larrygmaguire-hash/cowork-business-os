# Project Folder Template

This is the template structure used when creating a new project under a client. The `creating-projects` skill copies this folder when starting a new client project.

## Standard structure

```
P### Project Name/
├── CLAUDE.md           # Project-specific context
├── Research/           # Background reading and references
├── Drafts/            # Work in progress
└── Final/             # Completed deliverables
```

## Customising the template

To change the default structure for new client projects:
1. Edit the folder structure here
2. Update the CLAUDE.md content
3. Future projects will use the updated template
4. Existing projects are not affected

Keep the template generic. Project-specific content goes in the actual project folder, not here.
